Model Calculation Structure

The upgraded model first removes inflation, explains real gold with reserve and macro drivers, then converts the fitted value back to nominal dollars.

1. InputsGold, CPI, central-bank buying, US debt/GDP, consumer sentiment, broad dollar
2. DeflateTurn nominal gold into real gold using CPI
3. FitEstimate the real-gold center from standardized drivers
4. ReflateConvert fitted real gold back into nominal dollars
↓
Reserve demandCumulative central-bank gold buying
Fiscal pressureUS debt divided by US GDP
Risk pressureWeak consumer sentiment
Dollar pressureBroad dollar index

Key Formulas

Z(x) means the variable is standardized against the historical model sample.

Real gold = Nominal gold × CPI base / Current CPI
ln(Real gold) = 7.6642 + 0.2585×Z(CB cumulative buying) + 0.0438×Z(US debt/GDP) + 0.0143×Z(Weak consumer sentiment) - 0.1028×Z(Broad dollar)
Nominal fair value = exp(fitted ln real gold) × Current CPI / CPI base
Premium = Spot gold / Nominal fair value - 1
